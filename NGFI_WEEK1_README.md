## 📞 IF ISSUES OCCUR

**Email Bounced?**
- Log the bounce in daily report
- Mark prospect as "bounced" in database
- Remove from future sends
- Document reason (invalid, spam filter, etc.)

**No Opens After 48h?**
- Normal for cold email (opening happens 0-7 days)
- Keep monitoring
- Don't adjust

**Immediate Reply?**
- Respond same day if possible
- Note quality of response
- Log in daily report + database

**Email Rejected by System?**
- Check if email verification was wrong
- Try alternative contact method (if available)
- Log issue + move on
